# Wallpapers

This folder contains wallpaper images for the Windows 98 Display Properties.

## Available Wallpapers

The following wallpaper images are available in this directory:

- `jungle.png` - Lush jungle/forest scene with winding path
- `chia.png` - Chia branding wallpaper
- `orange-waves.png` - Abstract orange gradient waves
- `tanggang.life.png` - Tang Gang Life branding wallpaper
- `windows-98.png` - Classic Windows 98 logo with clouds
- `windows-orange.png` - Windows logo with orange gradient

## Image Requirements

- Format: PNG or JPG/JPEG supported
- Size: Recommended 1920x1080 or larger (will be scaled to fit)
- The app will use solid color backgrounds as fallback if images are not found

## Solid Color Options

Solid color wallpapers are built-in and don't require images:
- Teal (Classic) - #008080
- Orange - #ff6600
- Black - #000000
- Navy - #000080
