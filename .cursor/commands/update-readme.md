- After adding a new feature
- After changing dependencies
- After modifying Docker setup
- After changing build/deployment process
- After adding new configuration

## What this command does:
1. Reviews the current codebase changes
2. Updates README.md following README_WORKFLOW.md guidelines
3. Ensures Mac-focused instructions
4. Maintains Tang Gang vibes and beginner-friendly tone
5. Updates relevant sections (Features, Installation, Usage, Docker, etc.)
6. Verifies all commands work on Mac

- Use Cmd key shortcuts (not Ctrl)
- Reference Docker Desktop for Mac (if applicable)
- Keep Tang Gang energy and supportive tone
- Make it beginner-friendly for Abit

- Update README_UPDATE_CHECKLIST.md if needed
