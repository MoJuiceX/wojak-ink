# Update Docker Documentation

Update README.md with Docker-related changes.

## When to use:
- After modifying Dockerfile
- After changing docker-compose.yml
- After adding Docker features
- After testing Docker setup

## What to update:
1. **Running Locally Section**
   - Update Docker option instructions
   - Add/update docker-compose commands
   - Verify Mac-specific Docker Desktop references

2. **Docker Setup Section**
   - Update Docker configuration details
   - Update development commands
   - Update production build commands
   - Explain any new Docker files

3. **Prerequisites Section**
   - Ensure Docker Desktop for Mac is listed
   - Add installation link if needed

4. **Troubleshooting Section**
   - Add Docker-specific issues if found
   - Add solutions for common Docker problems

## Guidelines:
- Always reference "Docker Desktop for Mac"
- Test all docker-compose commands on Mac
- Include both development and production examples
- Explain Docker benefits clearly
- Keep beginner-friendly (Abit is new to Docker)

## Example updates:
- If adding docker-compose.yml: Document the new setup
- If changing ports: Update port references
- If adding volumes: Explain what they do
- If production build changes: Update build commands

Remember: Test Docker commands on Mac before updating README!
